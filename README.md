#  Number Guessing Game (Go)

## Beschrijving

Een simpele CLI-game in Go waarin je een getal moet raden tussen **1 en 100**.

##  Hoe werkt het?

* De computer kiest een willekeurig getal.
* Jij kiest een moeilijkheid:

  * Easy → 10 pogingen
  * Medium → 7 pogingen
  * Hard → 5 pogingen
* Na elke gok krijg je feedback

  * **Te hoog** of **te laag**
* Je wint als je het juiste getal raadt 
* Je verliest als je geen pogingen meer hebt 

## ▶ Starten

Open je terminal en run:

go run main.go

##  Vereisten

* Go geïnstalleerd (bijv. Go 1.18+)

##  Features

Random getal (1–100)
Moeilijkheidsniveaus
Feedback per gok
Teller voor pogingen
